# Kill window/pane in tmux

> <Prefix> & for killing a window
> 
> <Prefix> x for killing a pane
> 
> If there is only one pane (i.e. the window is not split into multiple panes, <Prefix> x would kill the window)
> 
> As always iterated, <Prefix> is generally CTRL+b. (I think for beginner questions, we can just say CTRL+b all the time, and not talk about prefix at all
